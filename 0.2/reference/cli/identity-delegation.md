# icp identity delegation

Manage delegations for identities

**Usage:** `icp identity delegation <COMMAND>`

###### **Subcommands:**

* [`request`](identity-delegation-request.md) — Create a pending delegation identity with a new P256 session key
* [`sign`](identity-delegation-sign.md) — Sign a delegation from the selected identity to a target key
* [`use`](identity-delegation-use.md) — Complete a pending delegation identity by providing a signed delegation chain
