# icp identity

Manage your identities

**Usage:** `icp identity <COMMAND>`

###### **Subcommands:**

* [`account-id`](identity-account-id.md) — Display the ICP ledger or ICRC-1 account identifier for the current identity
* [`default`](identity-default.md) — Display or set the currently selected identity
* [`delegation`](identity-delegation.md) — Manage delegations for identities
* [`delete`](identity-delete.md) — Delete an identity
* [`export`](identity-export.md) — Print the PEM file for the identity
* [`import`](identity-import.md) — Import a new identity
* [`link`](identity-link.md) — Link an external key to a new identity
* [`list`](identity-list.md) — List the identities
* [`new`](identity-new.md) — Create a new identity
* [`principal`](identity-principal.md) — Display the principal for the current identity
* [`rename`](identity-rename.md) — Rename an identity
