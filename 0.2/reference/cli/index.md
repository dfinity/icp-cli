**Usage:** `icp [OPTIONS] [COMMAND]`

###### **Subcommands:**

* [`build`](build.md) ‚Äî Build canisters
* [`canister`](canister.md) ‚Äî Perform canister operations against a network
* [`cycles`](cycles.md) ‚Äî Mint and manage cycles
* [`deploy`](deploy.md) ‚Äî Deploy a project to an environment
* [`environment`](environment.md) ‚Äî Show information about the current project environments
* [`identity`](identity.md) ‚Äî Manage your identities

###### **Options:**

* `--project-root-override <PROJECT_ROOT_OVERRIDE>` ‚Äî Directory to use as your project root directory. If not specified the directory structure is traversed up until an icp.yaml file is found
* `--debug` ‚Äî Enable debug logging

  Default value: `false`
* `--identity-password-file <FILE>` ‚Äî Read identity password from a file instead of prompting
