# icp canister snapshot

Commands to manage canister snapshots

**Usage:** `icp canister snapshot <COMMAND>`

###### **Subcommands:**

* [`create`](canister-snapshot-create.md) — Create a snapshot of a canister's state
* [`delete`](canister-snapshot-delete.md) — Delete a canister snapshot
* [`download`](canister-snapshot-download.md) — Download a snapshot to local disk
* [`list`](canister-snapshot-list.md) — List all snapshots for a canister
* [`restore`](canister-snapshot-restore.md) — Restore a canister from a snapshot
* [`upload`](canister-snapshot-upload.md) — Upload a snapshot from local disk
