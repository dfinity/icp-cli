# icp canister

Perform canister operations against a network

**Usage:** `icp canister <COMMAND>`

###### **Subcommands:**

* [`call`](canister-call.md) — Make a canister call
* [`create`](canister-create.md) — Create a canister on a network
* [`delete`](canister-delete.md) — Delete a canister from a network
* [`install`](canister-install.md) — Install a built WASM to a canister on a network
* [`list`](canister-list.md) — List the canisters in an environment
* [`logs`](canister-logs.md) — Fetch and display canister logs
* [`metadata`](canister-metadata.md) — Read a metadata section from a canister
* [`migrate-id`](canister-migrate-id.md) — Migrate a canister ID from one subnet to another
* [`settings`](canister-settings.md) — Commands to manage canister settings
* [`snapshot`](canister-snapshot.md) — Commands to manage canister snapshots
* [`start`](canister-start.md) — Start a canister on a network
* [`status`](canister-status.md) — Show the status of canister(s)
* [`stop`](canister-stop.md) — Stop a canister on a network
* [`top-up`](canister-top-up.md) — Top up a canister with cycles
